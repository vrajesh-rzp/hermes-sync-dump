# VRM Project Scrapped by Leadership

**Date:** 2026-06-17
**Decision:** Leadership has scrapped the VRM (Virtual Relationship Manager) project.
**Impact:** Vrajesh is no longer the PM for VRM. He is now solely the PM for Onboarding Infrastructure.

## Context

VRM was an AI-powered merchant account manager aimed at improving M3-M12 retention. The project was in early scaling phase (1,000 merchants live) when leadership decided to discontinue it.

## Outcome

- Vrajesh's sole focus is now **Onboarding Infra** pod
- VRM OKRs are no longer tracked
- Onboarding Infra OKRs (Compliance, API Decomp, Funnel Improvement, Merchant Experience) are the active priorities
